上述链接讲述了babel版本不一致以及如何配置babelrc

在配置babel的时候，会遇到几个问题。

  1. 可能没安装babel-cli
  2. babel各版本老旧或不兼容

目录结构

package.json

先安装webpack webpack-cli

npm install -D webpack webpack-cli

根据webpack版本安装babel。有时候babel版本之间会不兼容，所以babel的版本也要十分注意。

在webpack 4.2以上使用

npm install -D babel-loader @babel/core @babel/preset-env

index.html

上面index.html中引入的是经过babel转化后打包好的index-webpack.js

index.js

这里没什么好讲的，主要就是写了一些ES6的语法把。

.babelrc

这个文件需要放在根目录里，应该就是和package.json在同一个文件夹下吧

webpack.config.js

很简单，在module里设置规则，遇到js文件就用babel-loader解析

配置webpack的时候，要注意的就是路径一定要正确
