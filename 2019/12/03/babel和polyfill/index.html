Babel和polyfill

区别

babel：只转换最新的语法，不会转换最新的API，比如document.querySelector、Element.classList、ES5 新增的 Array 方法、Date.now、ES6 中的 Object.assign、Promise 等。

polyfill：能够提供新的API供老版浏览器使用，缺啥补啥，跟打补丁一样
