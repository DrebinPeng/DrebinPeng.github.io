箭头函数this指向

首先重要一点

箭头函数没有this，this来源于作用域链

什么意思？咱们看个例子

先来分析obj2.sayName()()为什么是shabi吧

首先：

obj2.sayName()执行时，在sayName中，this指向的是obj2，即下面这个例子，

然后返回一个匿名函数。下一个步骤是执行该匿名函数obj2.sayName()()

谁执行该匿名函数的？？

答案是：Window全局，因此this.name为shabi

再来分析下obj.sayName()()吧

这里就完美诠释了“箭头函数的this来源于作用域链”

首先执行obj.sayName()

同理，此时sayName中的this一样是指向obj

然后返回箭头函数，由于箭头函数没有this，this来源于作用域链，所以他沿着作用域链往上去找this

作用域链长这样

| - obj

​ | - name: rubin

​ | - sayName: this

​ | - 箭头函数

箭头函数往上追溯，便找到了sayName中的this，

所以此时this.name为rubin
