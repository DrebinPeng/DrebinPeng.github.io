三、制作气泡框的边框

如图，如果要设计一个气泡框，这个气泡框也有边框呢？

要点：

Boolean

数据类型 转换为true 转换为false
Boolean true false
Number 任何非零值 0和NaN
String 任何非空字符串 空字符串“”
Object 任何对象 null
Undefined （不存在这个情况） undefined

Number

这么大个字，肯定是数字咯

讲一下其他数据类型向Number是怎么转换的把

  • 如果Boolean，true变为1，false变为0

  • 如果Number，是啥就是啥

  • 如果null，那么值就是0

  • 如果undefined，那么返回NaN

  • 如果是字符串，遵循以下规则

    • 如果字符串只包含了数字（包含了正负号），则转换为十进制数字，其中前导零将被省略，比如011变为11

    • 如果是有效的浮点格式，那就返回浮点数咯

    • 如果是有效的十六进制格式，那就转为十进制数字

    • 如果为空字符串，就转为0

    • 其他格式一律是NaN

  • 如果是对象，则调用对象的valueOf函数来进行转换

传递参数

那么传递参数时，参数是怎么传递的呢？

这其实非常简单，一点都不绕。在《高程》中，说了一句

访问变量有按值和按引用两种方式，而参数只能按值传递

这是什么意思呢？

如果给参数传的是基本数据类型，这很好理解，就是复制一份同样的值给参数嘛，参数的改变不会修改原变量。

而如果给参数传递的是对象呢？

也很好理解，就是将引用复制一份给参数嘛，原变量和参数同时指向内存中的唯一一份数据

从上面的代码来看，即使obj2指向了新的对象，也没有影响到obj1的指向。

因为他们两个是独立的变量。

引用变量可以有无数个，他们可以都指向内存当中唯一一份实例对象数据。

由上面来看，传递参数就是按值访问的
